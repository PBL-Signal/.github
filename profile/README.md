# PBL-SIGNAL
SIGNAL Developers Team 

안녕하세요,    
Nodejs와 Redis를 이용하여 실시간 다중 플레이어 보안 교육 게임을 만든 SIGNAL 팀입니다.    

보안에 관심있는 자들이 누구나 쉽게 보안 교육을 할 수 있도록 정보보안 전공 학생 4명이 지도 교수 지도하에 게임을 기획 및 개발하였습니다.   
본 게임은 MITRE ATT&CK 매트릭스 기반의 공격 시나리오를 이용한 게임으로, Nodejs 서버와 WebGL로 구현되었습니다.   


자세한 게임 설명은 아래 게임 소개 페이지(Notion)을 참고하시기 바랍니다.
